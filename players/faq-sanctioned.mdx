ClassicUO Web is officially sanctioned for use on the live, retail Ultima Online servers managed by Broadsword. This page answers common questions from players who want to play on official servers using the web client. Visit [retail.classicuo.org](https://retail.classicuo.org) to get started.

<Accordion title="Can I use ClassicUO Web on the official Ultima Online servers?">
  Yes. Please also refer to the [official uo.com wiki](https://uo.com/wiki/classicuo-web-client/) for additional guidance provided directly by Broadsword.
</Accordion>

<Accordion title="Does the web client have an assistant like UOAssist?">
  Yes. ClassicUO Web includes its built-in Web Assistant. However, certain assistant features are restricted or disabled when playing on the official Ultima Online servers, based on Broadsword's preferences.
</Accordion>

<Accordion title="Why are certain features disabled on official servers?">
  Broadsword has chosen to restrict or disable features they consider to give players an unfair advantage. This restriction is a necessary condition of the web client being sanctioned for use on their servers. It is not something the ClassicUO team can change unilaterally.
</Accordion>

<Accordion title="What features does the web client have?">
  ClassicUO Web includes almost everything available in the desktop client — high FPS, zoom, unlimited window size, and more — as well as web-only features such as:

  - [Game Profiles](/features/game-profiles)
  - [Chat](/features/chat)
  - [Grid Containers](/features/grid-containers)
  - [Auto-loot](/features/autoloot)
  - [Action Bar](/features/action-bar)
  - [Hotkeys](/features/hotkeys)
  - [Tooltips](/features/tooltips)

  Some of these features may be restricted on official servers at Broadsword's discretion.
</Accordion>

<Accordion title="Is the web client safe to use?">
  Yes. Broadsword has access to the source code and has vetted the client as safe. Broadsword also remains aware of any ongoing changes to the web client.

  In some respects, the web client is *safer* than the desktop client: because it runs inside the browser sandbox, it cannot do anything harmful to your computer — just like any other website you visit.

  The ClassicUO team does not save any personally identifiable information and does not share your data with third parties. The only personal data stored is your Discord username and ID. You will only be contacted via the ClassicUO Discord server (for announcements) or within the web client itself.
</Accordion>

<Accordion title="Does the web client replace the standard retail clients?">
  No. ClassicUO Web is an alternative client and does not replace the standard Classic Client or Enhanced Client. Players may continue to use those clients as normal.
</Accordion>

<Accordion title="Is the ClassicUO team compensated by Broadsword?">
  No. The ClassicUO team receives no compensation in any form from Broadsword. The client is built in the team's free time as a passion project. The team has agreed to Broadsword's conditions — and the additional work that comes with being a sanctioned client — so that players can enjoy the web client without fear of repercussions.
</Accordion>

<Accordion title="Why is the web client allowed when the desktop version isn't?">
  The desktop version was deemed unsuitable by Broadsword because it can be easily modified to enable features that provide unfair advantages. The web client has built-in feature restrictions that Broadsword controls, ensuring only gameplay-fair features are available on their servers.
</Accordion>

<Accordion title="Can I bring my ClassicUO desktop settings into the web client?">
  Yes. You can import your desktop ClassicUO settings into the web client. Click on your display picture in the top right of the client and select the **"Import from desktop ClassicUO"** button.
</Accordion>
