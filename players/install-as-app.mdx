Modern browsers let you install any web page as a **Progressive Web App (PWA)** — a standalone application that gets its own window, taskbar icon, and desktop shortcut. For the ClassicUO Web Client, installing as an app provides three meaningful benefits over playing in a regular browser tab.

**Why install as an app?**

- **Standalone window** — no address bar, tab strip, or browser toolbar cluttering your screen.
- **Taskbar and desktop shortcut** — launch the game directly without opening a browser first.
- **Persistent storage** — installed PWAs request persistent storage from the browser. Chrome exempts persistent storage from automatic eviction, so your 3.2 GB of downloaded game files won't disappear when your disk runs low.
- **Fewer hotkey conflicts** — browser shortcuts like `Ctrl+W`, `Ctrl+T`, and `F5` won't interfere with in-game keybindings.

---

## Install on Chrome

<Steps>
  <Step title="Navigate to the game and wait for it to load">
    Open the ClassicUO Web Client URL in Chrome and wait for the initial load to complete. The install option only appears once the page has fully loaded.
  </Step>
  <Step title="Open the three-dot menu">
    Click the **three-dot menu** (⋮) in the top-right corner of the Chrome window.
  </Step>
  <Step title='Click "Install page as app…"'>
    Select **Install page as app…** from the menu. If this option does not appear, make sure you are on the correct URL and that the page has finished loading.
  </Step>
  <Step title="Confirm the installation">
    Click **Install** in the confirmation dialog that appears.

    Chrome will open the game in its own standalone window and automatically add shortcuts to your **taskbar** and **desktop**.
  </Step>
</Steps>

## Install on Microsoft Edge

<Steps>
  <Step title="Navigate to the game and wait for it to load">
    Open the ClassicUO Web Client URL in Edge and wait for the page to finish loading.
  </Step>
  <Step title="Open the three-dot menu and go to Apps">
    Click the **three-dot menu** (⋯) in the top-right corner of Edge, then hover over **Apps**.
  </Step>
  <Step title='Click "Install this site as an app"'>
    Select **Install this site as an app** from the submenu, then click **Install** in the dialog.
  </Step>
</Steps>

<Note>
  Microsoft Edge is an excellent choice for the game browser if you want to keep game and everyday browsing completely separate. See the [dedicated browser guide](players/troubleshooting-dedicated-browser) for a full walkthrough.
</Note>

## Your Settings Are Saved to the Cloud

<Tip>
  All in-game settings, keybind macros, and character profiles are saved to your account in the cloud — not to the browser. After installing as an app (or switching to any other browser), everything loads automatically on your first login. You don't need to reconfigure anything.
</Tip>

---

## Uninstalling the App

To remove the installed app, open it, click the three-dot menu inside the app window, and select **Uninstall**. This removes the shortcuts but does not delete your downloaded game files or account data. You can always reinstall by following the steps above.
