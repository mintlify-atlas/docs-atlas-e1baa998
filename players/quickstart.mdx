ClassicUO Web lets you jump into Ultima Online directly from your browser — no installer, no launcher, and no manual file management. This guide walks you through everything from system requirements to your first login.

## System Requirements

Before you begin, make sure your setup meets these requirements:

- **Browser:** Google Chrome or any Chromium-based browser (e.g. Edge, Brave, Opera). Firefox and Safari are **not** supported.
- **Disk space:** Approximately **3.2 GB** of free storage for the browser's Origin Private File System cache.
- **Internet connection:** A stable broadband connection. First-time downloads are larger; subsequent launches use cached files.

<Note>
  ClassicUO Web relies on APIs that are only available in Chrome and Chromium-based browsers. If you're on Firefox or Safari, you'll need to switch browsers before continuing.
</Note>

## Play on Free Shards

Use [play.classicuo.org](https://play.classicuo.org) to browse the community server list and join any free-to-play shard.

<Steps>
  <Step title="Open a Chromium-based browser">
    Launch Google Chrome, Microsoft Edge, Brave, or another Chromium-based browser. Make sure it is up to date for the best compatibility.
  </Step>
  <Step title="Visit play.classicuo.org">
    Navigate to [play.classicuo.org](https://play.classicuo.org). You'll land on the ClassicUO Web server list, where available shards are displayed.
  </Step>
  <Step title="Log in with Discord">
    Click the **Sign in with Discord** button. Discord authentication keeps the service secure, prevents abuse, and syncs your macros, scripts, and hotkeys across devices. You will only need to authorise once.
  </Step>
  <Step title="Pick a shard">
    Browse the server list and click on a shard that interests you. Each listing shows a description, player count, and other details to help you choose.
  </Step>
  <Step title="Click Play">
    Click the **Play** button on the shard's listing page to launch the client for that shard.
  </Step>
  <Step title="Wait for the initial download">
    The first time you play, the client will fetch and cache the game files needed for that shard. This typically takes **a few minutes** depending on your internet speed. Progress is shown on-screen. Subsequent launches are much faster as files are already cached locally.
  </Step>
  <Step title="Log in and play">
    Once loading is complete, the familiar Ultima Online login screen appears. Enter your shard account credentials and you're in.
  </Step>
</Steps>

## Play on Official Ultima Online Servers (Retail)

ClassicUO Web has been sanctioned for use on the official Ultima Online servers managed by Broadsword. Visit [retail.classicuo.org](https://retail.classicuo.org) to access the retail version of the client.

<Note>
  Some Web Assistant features are restricted on official servers at Broadsword's request. See the [Retail FAQ](/players/faq-sanctioned) for a full breakdown of what is and isn't available.
</Note>

## Storage & Caching

Game files are stored inside your browser's local storage using the Origin Private File System. You can view current disk usage by clicking the **profile icon** at the top right of the side menu inside the client. If you need to free up space, you can clear the storage from the same menu — note that this will require the files to be re-downloaded on your next session.
